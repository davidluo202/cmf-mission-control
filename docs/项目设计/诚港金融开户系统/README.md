## 🎯 开发优先级

1. **Phase 1** - 基础信息收集
   - Case 2: 公司基本信息
   - Case 3: 公司财务与投资概况
   - Case 4: 关联人士信息

2. **Phase 2** - 合规流程
   - Case 5: 风险评估问卷（基于CMF013）
   - Case 6-7: 银行账户与税务信息（复用现有组件）

3. **Phase 3** - 文档与签署
   - Case 8: 文件上传
   - Case 9: 监管声明与签署（展示账户协议）
   - Case 10: 预览与提交
