# 机构客户开户系统 - 开发任务清单

## 🎯 项目概览

**目标：** 为诚港金融开户系统增加机构客户在线开户功能  
**技术栈：** Next.js 14 + React + TypeScript + Ant Design/shadcn + MySQL

### 1.2 后端API开发

**RESTful API端点：**

#### 账户管理
- [ ] `POST /api/corporate-accounts` - 创建账户草稿
- [ ] `GET /api/corporate-accounts/:id` - 获取账户详情
- [ ] `PATCH /api/corporate-accounts/:id` - 更新账户信息
- [ ] `POST /api/corporate-accounts/:id/submit` - 提交申请
- [ ] `GET /api/corporate-accounts` - 列表查询（管理后台）

#### Case数据保存
- [ ] `PATCH /api/corporate-accounts/:id/basic-info` (Case 2)
- [ ] `PATCH /api/corporate-accounts/:id/financial-profile` (Case 3)
- [ ] `POST /api/corporate-accounts/:id/connected-parties` (Case 4)
- [ ] `PATCH /api/corporate-accounts/:id/connected-parties/:partyId`
- [ ] `DELETE /api/corporate-accounts/:id/connected-parties/:partyId`
- [ ] `POST /api/corporate-accounts/:id/risk-assessment` (Case 5)
- [ ] `POST /api/corporate-accounts/:id/bank-accounts` (Case 6)
- [ ] `PATCH /api/corporate-accounts/:id/tax-info` (Case 7)

#### 文件上传
- [ ] `POST /api/corporate-accounts/:id/documents` - 上传文件
- [ ] `GET /api/corporate-accounts/:id/documents/:docId` - 下载文件
- [ ] `DELETE /api/corporate-accounts/:id/documents/:docId` - 删除文件

#### 签署与提交
- [ ] `POST /api/corporate-accounts/:id/declarations` - 保存签署信息
- [ ] `POST /api/corporate-accounts/:id/finalize` - 最终提交

**技术要点：**
- 文件存储：AWS S3 / 阿里云OSS / 本地文件系统
- 文件大小限制：5MB
- 支持格式：PDF, JPG, PNG
- 数据验证：使用Joi或Zod
- 错误处理：统一错误码体系

**交付物：**
- API文档（Swagger/OpenAPI）
- 单元测试（Jest）
- Postman测试集合

**负责人：** 后端开发  
**工时估算：** 5-7天
