# 参考之前系统的设计笔记

## 界面风格观察

### 首页设计
- **标题**: 誠港金融開戶系統 / Account Opening System
- **副标题**: 為香港SFC持牌法團設計的完整、合規、現代化的客戶開戶申請系統
- **颜色方案**: 
  - 主色调: 蓝色 (按钮和标题使用鲜明的蓝色)
  - 背景: 白色/浅色背景
  - 文字: 黑色/深灰色
- **布局**: 居中对称布局，简洁现代

### 核心功能模块
1. SFC合规设计 - 最核心遵守SFC监管要求，内置KYC/AML合规检查
2. 现代化交互 - 采用新的KYC技术，支持人脸识别、数字签名等
3. 完整的用户旅程 - 从个人资料到开户流程，流畅的用户体验
4. 后台管理系统 - 完整的审批工作流、合规和文档管理平台

### 语言支持
- 中英文双语界面
- 右上角有语言切换按钮

## 需要改进的问题（根据知识库）
1. 数据持久性问题 - Case 8、9、10等页面数据丢失
2. 复杂的数据存储逻辑导致错误
3. PDF预览中文显示问题

## 新系统改进方向
1. 采用更简单、更可靠的数据存储逻辑（类似Case 3-6的方式）
2. 确保所有页面数据持久化到数据库
3. 使用支持中文的PDF生成方案
4. 严格的表单校验规则
5. 清晰的用户流程导航
