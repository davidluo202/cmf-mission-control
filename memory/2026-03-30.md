## Mission Control v0.5.2 — 主动巡查 + 告警升级系统

### 背景
- David 要求 MC 建立双向互动闭环：Agent→MC 上报 + MC→Agent 主动巡查 + MC→CEO/David 告警
- 旧的 Keepalive 机制掩盖了真实离线状态，已移除

### 新增功能
- **主动巡查循环**: 每5分钟运行，检测 stale agents、health check 异常
- **告警系统**: alerts 表，severity=INFO/WARNING/CRITICAL，target=ICY/DAVID/ALL
- **升级规则**: 30min stale→WARNING(Icy), 2h offline→CRITICAL(All), 多Agent离线→CRITICAL(David)
- **Alerts 页面**: /alerts，按严重性排序，一键确认，批量确认
- **Header 通知铃铛**: 实时角标，下拉显示最近告警
- **Dashboard CRITICAL 横幅**: 有 CRITICAL 告警时顶部红色横幅
- **Agent 上报协议**: SPEC 第7节，15分钟状态SLA，30分钟健康检查SLA
- Commit: `e5a0037`, 已推送 main
